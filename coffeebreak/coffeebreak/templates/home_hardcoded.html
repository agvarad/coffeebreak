A scoop of Django and Google App Engine



Django, one of the most popular amongst all the python frame-works is used in building website.

Pre-requisites

1. Python 2.7+

2. Google App Engine Launcher

3. Google App Account


Build your Appengine App

1. Create your project using django-admin.py startproject .


django-admin.py startproject "Projectname"

2. Create a file "app.yaml" in the project directory.



version: 1

runtime: python27

api_version: 1

threadsafe: true



libraries:

- name: django

   version: "1.4"



builtins:

- django_wsgi: on



handlers:

- url: /static/admin

   static_dir: static/admin

   expiration: '0'

3. Edit the urls.py

4. Create a view to render the response - views.py

5. Modify settings.py to include template - Add to TEMPLATE_DIRS

5. Create space for your static files. Create a folder static and place your static files in this directory.

5. Download your SDK for Google App Engine and run in command line. In my case its installed in "/usr/local" directory

6. Start your application to test if responses are rendered as expected.

7. Create a Google App Account and create a project with the same name as your application. Remember this unique name is used to generate your app's url.

8. Host your application in Google cloud using appcfg.py.

9. http://yourprojname.appspot.com.

10. Login to your Google App Engine to see your analytics.
